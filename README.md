# To run project

* Ruby version - ruby 2.3.3

`` $ rails db:create``

`` $ rails db:migrate``

`` $ rails db:seed``

`` $ rails s``

Must have TCP connection, in order for queue to work
To send products 

 `` $ ruby push_to_queue.rb ``
 
To remove products

 `` $ ruby remove_from_queue.rb ``
